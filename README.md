# ProjectTracker

Проєкт для відстеження робочих проєктів і задач українською мовою.

## Опис

`ProjectTracker` — це простий консольний трекер проєктів на Python. Програма дозволяє створювати проєкти, додавати до них задачі, відслідковувати статус і дедлайни, а також бачити прострочені елементи.

## Можливості

- Додавання/редагування проєктів
- Додавання/редагування задач у проєктах
- Вибір статусу проєкту: Активний, На паузі, Завершено, Скасовано
- Вибір статусу задачі: Очікує, В процесі, Виконано
- Перевірка прострочених дедлайнів
- Збереження даних у файл `ProjectTracker.json`

## Встановлення

1. Встановіть Python 3.x
2. Скопіюйте `ProjectTracker.py` і `ProjectTracker.json` в одну папку

## Використання

```bash
python ProjectTracker.py
```
Файли
ProjectTracker.py — основний скрипт
ProjectTracker.json — файл для збереження даних
